HTML

What is HTML?
HTML stands for Hypertext Markup Language, and is used to create about 96% of all sites. Even FlowScripts (this website) is coded in HTML! Everything is made up of elements, they include a start tag, content, and an end tag.

Example: <b> Text </b>
The text in the <b> tags will be bold.

CSS

CSS (Cascading Style Sheets), is used to style your website, and make it eye-catching.
text color:
<p style=" color: blue; "> Text </p>
You can use these colors: red, black, blue, green, yellow, white, purple, pink, (and more!).
